## 🚀 Welcome to MouseX!

We are excited to announce the official **v1.0.0** release of **MouseX**! 

If you are tired of bulky mouse software that requires cloud logins, telemetry tracking, and constant internet connectivity just to change a button mapping, MouseX is built for you. It is a lightweight, local-first Windows application that runs quietly in your system tray, letting you map your mouse buttons and scroll wheels directly to keyboard keys and repeating macros.

---

## ✨ Key Features in v1.0.0

### 🖱️ Map Every Mouse Action
Customize all **7 input channels** on your mouse:
- Left Click (Button 1)
- Right Click (Button 2)
- Middle Click / Wheel Click (Button 3)
- Back Button (Button 4)
- Forward Button (Button 5)
- Scroll Wheel Up (Button 6)
- Scroll Wheel Down (Button 7)

> **How it works:** When a mapping is active, the native mouse event is completely blocked from propagating to other apps (preventing accidental clicks or navigation page-back triggers) and is replaced by your custom keyboard input.

### ⌨️ Multi-Key Chords & Sequences
Map a single mouse action to up to **3 keyboard keys**:
- **Chord Mode**: Simulates pressing all keys simultaneously (e.g., mapping Middle Click to `Ctrl + Shift + Esc` to open Task Manager).
- **Sequence Mode**: Taps the mapped keys one after another in rapid succession.

### 🔁 Continuous Repeating (Turbo Macros)
Need to trigger a key repeatedly? Customize your repeat options for buttons 1–5:
- **Hold to Repeat**: Keep the mouse button pressed down to continuously fire the key at your selected speed.
- **Toggle Mode (Repeat Until Click)**: Tap the mouse button once to start a continuous key loop, and tap it again to turn it off.
- **Speed Slider**: Configure delays anywhere from **10 milliseconds** (ultra-fast 100 triggers/sec) up to **1000 milliseconds** (1 trigger/sec).

### 🎯 Smart App-Specific Profiles
Your profiles change automatically depending on what you are doing:
- Create separate configurations for different applications (e.g., a gaming macro for `valheim.exe` and tab-scrolling for `chrome.exe`).
- MouseX detects which window is currently active and swaps profiles instantly.
- Reverts back to your **Default** profile when you return to your desktop.

### 📥 System Tray & Windows Autostart
- **Tray Minimization**: Clicking "Close" on the dashboard window minimizes the app to the system tray, keeping your remappings running.
- **Autostart on Boot**: Check "Autostart on Boot" in the dashboard, and MouseX will register in the Windows Registry to load automatically when you log in.
- **Quick Controls**: Right-click the tray icon to toggle the mapping hook on/off or exit the app completely.

### 🎨 Modern & Premium User Interface
- Beautiful custom dark theme with deep-indigo and violet-glow accents.
- Responsive design that centers automatically and adjusts to your screen size.
- Checkboxes styled as modern, physical sliding toggle switches.

## 🛠️ System Requirements & Running

- **Operating System**: Windows 10 or Windows 11 (64-bit).
- **Java Runtime**: You must have Java installed on your computer. Make sure `%JAVA_HOME%` or your Java system path is configured so the app can launch the Java Virtual Machine.

### How to Run:
1. Download **`MouseX.exe`** from the [GitHub Releases](https://github.com/Felix-au/MouseX-Absolute-Mouse-Control/releases) page on the [MouseX Repository](https://github.com/Felix-au/MouseX-Absolute-Mouse-Control).
2. Place it in a folder of your choice (it will generate a `profiles.json` file in that folder to save your settings).
3. Double-click **`MouseX.exe`** to start.
4. Set up your mappings, click **START HOOK** in the footer, and you're good to go!

## 💡 Quick Examples to Try

Here are some helpful mappings to get you started:

*   **Quick Shift-Click**: Map Mouse Button 4 (Back) to `Shift`. Perfect for shooters or productivity work.
*   **Browser Tab Cycler**:
    *   Map Scroll Up (Button 6) to `Tab`
    *   Map Scroll Down (Button 7) to `Shift + Tab` (as a Chord)
    *   *Result: Easily scroll through tabs using your mouse wheel!*
*   **Rapid Fire Jumper**: Map Middle Click to `Space` with `Enable repeat` turned on and set to `50ms`. Hold down the wheel to jump continuously in games.
